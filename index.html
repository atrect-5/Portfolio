Proyectos en los que he trabajado

Proyecto destacado:

Desarrollado completamente por mí como freelance.

Software de control y monitoreo remoto para maquinas de refrigeracion industrial.

Desarrollado para: SOS Servicios o Soluciones Integrales, como complemento a las máquinas de refrigeración que fabrican.

Tecnologías utilizadas:

  • Frontend: Aplicación desarrollada con Vite + React y empaquetada como aplicación de escritorio usando Electron.
  • Backend: Construido con Node.js y Express.js, y desplegado en Heroku para manejar la lógica y el procesamiento de datos.
  • Bases de datos: Uso de MongoDB para datos estructurados e InfluxDB para lecturas en tiempo real de las máquinas.
  • Comunicación: Comunicación segura mediante HTTPS con el backend y MQTTS para la conexión en tiempo real con las máquinas.
